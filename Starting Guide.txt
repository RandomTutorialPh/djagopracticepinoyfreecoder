Creating Env - python -m venv myenv
Running env - myenv\Scripts\activate
Deactivate env - deactivate

Exporting dependencies -pip freeze > requirements.txt
Reinstalling dependencies to other computer - pip install -r requirements.txt

install django - pip install django
install mysqlclient - pip install mysqlclient

creating first project on django - django-admin startproject (nameofproject) .
Run python project - python manage.py runser
Run python project thru hotspot or wifi - get first the ipv4 of your wifi. go to your django project - settings.py - on ALLOWED_HOSTS = [] add your ipv4. run this command "python manage.py runserver 192.168.56.1:8000" 

Git Guide
git push origin master --force - remove the current files on repo and upload the new files.

git init                            # Initialize a git repo (if not done yet)
git add .                           # Add all files
git commit -m "Initial commit"      # Commit the files
git branch -M main                  # Rename default branch to main
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git
git push -u origin main             # Push to GitHub
